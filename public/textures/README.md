# 纹理资源说明

这个目录用于存放太阳系天体的纹理图片。

## 所需纹理文件

为了获得最佳视觉效果，建议添加以下纹理图片（文件名需与代码中引用的一致）：

- `sun.jpg` - 太阳纹理
- `mercury.jpg` - 水星纹理
- `venus.jpg` - 金星纹理
- `earth.jpg` - 地球纹理
- `mars.jpg` - 火星纹理
- `jupiter.jpg` - 木星纹理
- `saturn.jpg` - 土星纹理
- `saturn-ring.jpg` - 土星光环纹理
- `uranus.jpg` - 天王星纹理
- `neptune.jpg` - 海王星纹理
- `moon.jpg` - 月球纹理

## 纹理资源获取

您可以从以下来源获取高质量的行星纹理：

1. **NASA 官网** - 提供大量真实的行星图像
   - https://www.nasa.gov/multimedia/imagegallery/

2. **Solar System Scope** - 提供免费的行星纹理
   - https://www.solarsystemscope.com/textures/

3. **Celestia Motherlode** - 天文模拟资源库
   - https://www.celestiamotherlode.net/

## 临时解决方案

如果暂时没有纹理图片，代码已经配置了 fallback 颜色，应用仍然可以正常运行，只是天体将显示为纯色球体。

您也可以使用简单的程序化纹理或纯色图片作为临时替代。
